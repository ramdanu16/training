The HTML Form Elements

Elemen HTML <form> dapat berisi satu atau lebih elemen formulir berikut:

  • <input>
  • <label>
  • <select>
  • <button>
  • <option>

The <input> Element

Salah satu elemen form yang paling banyak digunakan adalah elemen <input>.

Elemen <input> dapat ditampilkan dalam beberapa cara, bergantung pada atribut type.

Berikut beberapa contohnya:

Type Description Result
< input type="text" > Menampilkan kolom input teks satu baris
<input type="radio"> Menampilkan tombol radio (untuk memilih salah satu dari banyak pilihan)


<input type="checkbox"> Menampilkan kotak centang (untuk memilih nol atau lebih dari banyak pilihan)

<input type="submit"> Menampilkan tombol kirim (untuk mengirimkan formulir)



<input type="button"> Menampilkan tombol yang dapat diklik



Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman yang di targetkan di form "action"

<input type="password"> Karakter dalam bidang password ditutupi (ditampilkan sebagai tanda bintang atau lingkaran).

The <label> Element

Elemen <label> mendefinisikan label untuk beberapa elemen formulir.

Elemen <label> berguna bagi pengguna pembaca layar, karena pembaca layar akan membacakan label dengan lantang saat pengguna fokus pada elemen masukan.

Elemen <label> juga membantu pengguna yang mengalami kesulitan mengklik wilayah yang sangat kecil (seperti tombol radio atau kotak centang) - karena ketika pengguna mengklik teks dalam elemen <label>, tombol radio/kotak centang akan dialihkan.

Atribut for pada tag <label> harus sama dengan atribut id pada elemen <input> untuk mengikat keduanya.

Contoh-Contoh elemen label sudah di terapkan di elemen form.

The <select> Element And The <option> Element

Elemen <select> mendefinisikan daftar drop-down.

Elemen <option> mendefinisikan opsi yang dapat dipilih.

Secara default, item pertama dalam daftar drop-down dipilih.

Untuk menentukan opsi yang telah dipilih sebelumnya, tambahkan atribut selected ke opsi:

Information Result
Elemen Select dengan option yang biasa
Elemen Select dengan penambahan atribut selected ke element option yang tertentu
Nilai yang Terlihat (Gunakan atribut size untuk menentukan jumlah nilai yang terlihat)
